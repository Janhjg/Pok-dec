# Objetos de una Pokédex Objetos Principales
1. Pokémon

* id (número)
* nombre
* tipos (array)
* imagen/sprite
* descripción

2. Estadística

* hp (puntos de vida)
* ataque
* defensa
* ataque especial
* defensa especial
* velocidad

3. Tipo

* nombre (fuego, agua, planta, etc.)
* Mcolor
* icono
* fortalezas
* debilidades

4. Evolución

* pokémon anterior
* pokémon siguiente
* nivel de evolución
* método de evolución

5. Habilidad

* nombre
* descripción
* efecto

6. Movimiento/Ataque

* nombre
* tipo
* poder
* precisión
* puntos de poder

7. Generación

* número (1-9)
* región
* pokémon incluidos

8. Región

* nombre (Kanto, etc.)
generación
